# 更新日志

## [1.0.0] - 2025-02-09

### 新增功能
- 🎉 首次发布
- ✅ 支持 7 种波形布局模式（对称、左右、锚点、水平、圆形、弧形、螺旋）
- ✅ 15+ 精美的预设风格
- ✅ 完全自定义的条纹配置（颜色、间距、圆角、阴影等）
- ✅ 流畅的波形动画效果
- ✅ 支持多种手势交互（点击、滑动、长按）
- ✅ 内置音频录制和播放功能
- ✅ **新增** WaveformImageDrawer - 从音频文件生成静态波形图像
- ✅ **新增** LSWaveformAnalyzer - 音频样本分析器
- ✅ 支持 Swift 6.0 和 iOS 13+
- ✅ 同时支持 Swift Package Manager 和 CocoaPods 安装

### 技术特性
- 基于 Accelerate 框架优化，性能卓越
- 纯 Swift 实现，无外部依赖
- 完整的类型安全和并发安全支持
- 支持同步和异步 API

### 已知问题
- 无

### 兼容性
- iOS 13.0+
- Xcode 16.0+
- Swift 6.0+
